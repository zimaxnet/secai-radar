# Public Web App

Public-facing web application for secairadar.cloud/mcp

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Routes
- `/mcp` - Overview Dashboard
- `/mcp/rankings` - Rankings Dashboard
- `/mcp/servers/{slug}` - Server Detail
- `/mcp/daily/{date}` - Daily Brief
- `/mcp/methodology` - Methodology
- `/mcp/submit` - Submit Evidence
