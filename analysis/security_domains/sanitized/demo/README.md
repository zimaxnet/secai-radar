## Demo Walkthrough Artifacts

This subfolder holds the sanitized companion files that illustrate the assessment deliverables end to end:

- `contoso_Questionaire.txt` – excerpt of the generated questionnaire.
- `sp_contoso_Report.txt` – excerpt of the sanitized Word report.

Pair these with the sanitized workbook/report stored one directory up when running `build_domain_csvs.py` and the evidence collector. Replace the samples with up-to-date sanitized exports whenever you refresh the demo.
