SecAI Radar

Entra Agent ID, Registry & Foundry Control Plane Integration

Enterprise-Grade Agent Governance, Observability, and Security

Executive Summary

SecAI Radar has been enhanced with Microsoft's enterprise agent governance stack, transforming it from a standalone assessment tool into an enterprise-ready, agent-governed security platform. This integration adds unique agent identities, centralized inventory, comprehensive observability, and built-in security controls.

NEW ENTERPRISE SECURITY

Phase 1: Entra Agent ID Integration

Every AI agent in SecAI Radar now has a unique Entra Agent ID, providing enterprise-grade identity management similar to user identities in Microsoft Entra ID.

Key Features

Unique Identity

  • Each agent has unique Entra Agent ID
  • Service principal per agent
  • Lifecycle management
  • Identity validation

Authentication

  • Token-based authentication
  • Automatic token refresh
  • Per-agent, per-scope caching
  • Secure credential management

Access Control

  • Granular permissions
  • Blueprint-based policies
  • Tool authorization
  • Resource scoping

Auditing

  • Action-level auditing
  • Compliance reporting
  • Security monitoring
  • Audit trail

Phase 2: Agent Registry

The Agent Registry serves as the single source of truth for all AI agents, providing centralized inventory, discoverability, and governance.

Registry Features

Centralized Inventory

  • Complete agent catalog
  • Status tracking
  • Activity monitoring
  • Metadata management

Collections

  • Organize agents into groups
  • Built-in collections (quarantine)
  • Custom collections
  • Collection-based filtering

Quarantine

  • Isolate untrusted agents
  • Prevent discovery
  • Security review workflow
  • Unquarantine process

Discoverability

  • Agent-to-agent discovery
  • Human discovery
  • Search and filter
  • Capability-based search

API Endpoints

GET /api/registry/agents # List all agents GET /api/registry/agents/{agentId} # Get agent details POST /api/registry/agents # Register new agent PUT /api/registry/agents/{agentId}/status # Update agent status POST /api/registry/agents/{agentId}/quarantine # Quarantine agent DELETE /api/registry/agents/{agentId}/quarantine # Unquarantine agent

Phase 3: Foundry Control Plane

The Foundry Control Plane provides comprehensive observability, evaluation, and security controls for all AI agents.

Observability

OpenTelemetry

  • Distributed tracing
  • Performance metrics
  • Structured logging
  • Azure Monitor integration

Metrics Tracked

  • Response time (ms)
  • Token usage
  • Tool call counts
  • Success/failure rates

Evaluation Scores

  • Groundedness (0-1)
  • Task adherence (0-1)
  • Tool accuracy (0-1)
  • Relevance (0-1)

Real-time Monitoring

  • Live activity feed
  • Performance dashboards
  • Alert notifications
  • Cost tracking

AI-Powered Evaluators

Continuous evaluation using AI-powered evaluators ensures agent quality and detects behavioral drift:

  • Groundedness Evaluator - Verifies responses are grounded in provided context
  • Task Adherence Evaluator - Checks agent followed assigned task instructions
  • Tool Accuracy Evaluator - Validates tool call correctness
  • Relevance Evaluator - Assesses response relevance to query

Guardrails & Content Safety

Input Validation

  • Prompt injection detection
  • Length limits
  • Content safety checks
  • Pattern matching

Output Filtering

  • PII detection & redaction
  • Sensitive data masking
  • Content safety filtering
  • Automatic sanitization

Tool Authorization

  • Per-agent tool permissions
  • Blueprint-based policies
  • Violation logging
  • Automatic blocking

Rate Limiting

  • Per-minute limits (60 req/min)
  • Per-hour limits (1000 req/hr)
  • Per-agent enforcement
  • Automatic throttling

Phase 5: Dashboard & UI/UX

New dashboards and UI components provide comprehensive visibility and control over agent operations.

Agent Command Center

The main dashboard for agent governance and monitoring:

  • Agent Health Overview - Real-time status of all agents
  • Activity Feed - Live agent actions and tool calls
  • Performance Metrics - Response times, token usage, success rates
  • Evaluation Scores - Groundedness, task adherence, tool accuracy
  • Security Alerts - Content safety violations and anomalies

Agent Registry View

Centralized inventory with filtering and management:

  • Filter by status, collection, blueprint, or capability
  • View agent details and capabilities
  • Manage agent status and collections
  • Quarantine/unquarantine agents

Agent Detail View

Individual agent profile and monitoring:

  • Agent profile and identity information
  • Activity history with traces
  • Evaluation score trends
  • Configuration and guardrails
  • Tool usage breakdown

Observability Dashboard

Comprehensive observability and monitoring:

  • Trace Explorer - End-to-end request traces
  • Metric Charts - Time-series performance data
  • Evaluation Results - Quality scores over time
  • Cost Tracking - Token usage and estimated costs
  • Drift Detection - Behavioral drift alerts
