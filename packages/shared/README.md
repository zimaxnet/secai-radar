# Shared Package

Shared TypeScript types, schemas, and utilities

## Contents
- Type definitions (dataModel, mcp, graph, automation)
- Copy system (labels, disclaimers, badges)
- Utilities (canonicalIds, dedupe, calculations, feeds, socialTemplates, analytics)

## Usage
```typescript
import { MCPServerRecord } from '@secai-radar/shared/types/mcp'
import { getTierLabel } from '@secai-radar/shared/copy'
```
